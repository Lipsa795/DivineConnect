# 🕉️ DivineConnect

A full-stack spiritual platform that connects devotees with temples, pooja services, and charitable causes. Book poojas, donate to temples, order pooja samagri, and discover nearby temples using Google Places API.

---

## ✨ Features

* **🔐 User Authentication** - Secure signup/login with JWT
* **🙏 Pooja Booking** - Book various poojas with Razorpay test payment integration
* **❤️ Charity Donations** - Donate to temple causes (temple maintenance, food donation, education, medical aid)
* **🌸 Pooja Samagri** - Order pooja essentials with doorstep delivery
* **📍 Temple Finder** - Find nearby temples using real-time location with Google Places API
* **🗺️ Google Maps Integration** - Get directions to temples
* **⭐ Ratings & Reviews** - View temple ratings from Google
* **📱 Fully Responsive** - Works on desktop, tablet, and mobile

---

## 🛠️ Tech Stack

### Frontend

* React 18
* React Router DOM
* Tailwind CSS
* Axios
* Razorpay Test Integration

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* JWT Authentication
* Razorpay API
* Google Places API

## 📋 Prerequisites

Before running this project, make sure you have:

* Node.js (v14 or higher)
* MongoDB Atlas account (free tier)
* Razorpay test account
* Google Cloud Platform account with Places API enabled

## 🚀 Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/Lipsa795/DivineConnect.git
cd DivineConnect
```

---

## ▶️ How to Run the Project

### 📌 1. Open Terminal

* Press **Ctrl + Shift + `** in VS Code

---

### ⚙️ 2. Run Backend Server

```bash
cd backend
node server.js
```

---

### 💻 3. Run Frontend

Open a **new terminal** and run:

```bash
cd frontend
npm start
```

---

### 🌐 4. Access the Application

* Frontend: http://localhost:3000
